# README_generator

## Discription

This app creates a README file from user inputs in the command prompt. 

## Installation

This app requires inquirer and therefor the user must install node_modeules prior to use.

## Usage

Run in terminal using node. Fill in the prompt and press 'enter' when you are finished.

If you do not want to input for a ceratain section leave it blank and it will nothing will be rendered on the final README for that section.
